# Work detail media

One folder per stint id (matches `id` in `work-timeline.tsx`):

- `liquide/` — Liquide role
- `hopstack/` — Hopstack role
- `state-plate/` — The State Plate role

Drop images or videos here. Supported: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`, `.mp4`, `.webm`, `.mov`.

**Naming tips**

- `before.png` + `after.png` — shown side by side
- `1.jpg`, `2.png`, … — numbered gallery order
- `video.mp4` — embedded with controls

Files appear automatically on the work detail page (no code changes needed).
