# Courses & certifications (Resume app)

Add cover images here. Filename must match the `id` in `components/apps/resume/data.ts` (`LEARNING_ITEMS`).

Supported extensions: `.png`, `.jpg`, `.jpeg`, `.webp` (`.png` is tried first).

Examples:

- `google-ad-manager.png` or `google-ad-manager.jpg`
- `meta-ads-manager.jpg`
- `google-analytics.png`

Recommended: landscape **16:9**, at least **640×360** px (card thumbnail).
